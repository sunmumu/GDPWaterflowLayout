## Use
法一 cocopods引用
pod 'GDPWaterflowLayout'

法二 文件拖入项目中
1. 下载GDPWaterflowLayout框架,把GDPWaterflowLayout文件夹拷贝到项目中. 
2. 在控制器中#import "GDPWaterflowLayout.h".
